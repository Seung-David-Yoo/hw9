Summary: Latitude vs. X

The purpose of this project was to analysis how weather patterns may changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matploitlb to plot various aspects of the weather vs. latitude. Factors we look at included: termperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizatinos created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
